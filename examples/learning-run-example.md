# Example · Concept Learning Run

**goal**: Learn a new concept before turning it into review cards.

## Chain

```text
Intake -> Context -> Radar -> Normalize -> Learning -> Test -> Score -> Check -> Sediment
```

## Why

Do not send unfamiliar concepts directly into review or battle. First teach, enrich, test, score, then sediment.

## Required Enrichment Fields

```text
key_points
examples
misconceptions
applications
```

## Graduation

Only after enrichment and check pass:

```text
concept note -> review cards -> spaced retrieval
```
