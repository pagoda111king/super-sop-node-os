# Node Protocol Reference

Every node artifact should include:

```markdown
# <Node>

## Purpose
## Inputs
## Work
## Outputs
## Check
## Next
```

Check results:

```text
pass, fix, return, escalate, kill, accept_risk
```

Score verdicts:

```text
adopt, revise, monitor, reject
```

Sediment requires:

```text
source_ref
next_use_case
check status
downstream user or owner
```
