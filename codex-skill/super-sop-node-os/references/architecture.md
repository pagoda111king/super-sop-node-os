# Architecture Reference

## 6 Layers

1. Input Capture
2. Context & Memory
3. Cognition & Orchestration
4. Action
5. Validation & Governance
6. Sediment Assets

## 14 Nodes

| Node | Role |
|---|---|
| Intake | messy input -> task object |
| Context | task + history -> context pack |
| Radar | question -> traceable evidence |
| Normalize | messy evidence -> structured records |
| Data | query need -> schema/storage |
| Planning | goal -> steps/dependencies |
| Routing | state -> path |
| Learning | concept -> teachable lesson |
| Action | plan -> real action |
| Test | claim -> reproducible evidence |
| Score | evidence + tests -> verdict |
| Check | artifact -> pass/fix/return |
| Handoff | artifact -> person/system/agent |
| Sediment | experience -> reusable asset |
