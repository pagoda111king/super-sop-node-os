# Nodeize Task

Convert the user's request into a Super SOP Node OS run.

Use this structure:

```text
1. Restate the goal.
2. Select the minimal node chain.
3. Create a run_id.
4. Execute nodes in order.
5. Produce artifacts.
6. Run Test/Score/Check when needed.
7. Sediment reusable learnings.
```

Default chains:

```text
Research: Intake -> Context -> Radar -> Normalize -> Score -> Check -> Sediment
Implementation: Intake -> Context -> Planning -> Routing -> Action -> Test -> Check -> Handoff -> Sediment
Learning: Intake -> Context -> Radar -> Normalize -> Learning -> Test -> Score -> Check -> Sediment
```
